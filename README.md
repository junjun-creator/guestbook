## GuestBook
  간단한 방명록 남기기, 방문횟수 측정하기, 관리자모드(방명록 삭제기능) 등을 구현한 웹 프로그램

---
### 🧒Author
  - [김병준](http://github.com/junjun-creator)

### 🤹‍♂️구현 기능
  - [x] 방명록 등록
  - [x] 쿠키(Cookie)를 활용한 방문횟수 측정
  - [x] 세션(Session)을 활용한 관리자로그인
  - [x] 방명록 삭제
  - [x] 페이저(pager) 기능
